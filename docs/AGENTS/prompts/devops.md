# Prompts para Tareas de DevOps

## Añadir un nuevo script a package.json

"Añade un nuevo script en `Daring Dungeon Delver/package.json` llamado `generate-assets`. Este script debe ejecutar todos los scripts de Python para la generación de atlas en secuencia: `generate_atlas.py`, `generate_hero_atlas_normalized.py`, y `generate_atlas_centered.py`. Deben ejecutarse con el intérprete de python."

## Modificar la configuración de Vite

"Modifica el archivo `Daring Dungeon Delver/vite.config.ts`. Añade un alias de ruta para que `@/game` resuelva a la ruta `src/game`. Asegúrate de que los alias existentes sigan funcionando."
