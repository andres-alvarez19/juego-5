# Guía para Agentes de IA - Daring Dungeon Delver

Este documento proporciona el contexto y las directrices para los agentes de IA que trabajan en el proyecto **Daring Dungeon Delver**. El objetivo es asegurar que las contribuciones sean consistentes, de alta calidad y alineadas con la arquitectura y convenciones del proyecto.

## Misión del Proyecto

Daring Dungeon Delver es un juego de exploración de mazmorras 2D. Tu rol como agente es asistir en el desarrollo, mantenimiento y mejora del juego, siguiendo las guías establecidas en esta documentación.

## Cómo Usar esta Documentación

Consulta las diferentes secciones para entender la arquitectura, el flujo de trabajo, las guías de estilo y otros aspectos cruciales del proyecto antes de generar o modificar código. Los prompts de ejemplo en la sección `prompts/` te servirán como base para tareas comunes.
